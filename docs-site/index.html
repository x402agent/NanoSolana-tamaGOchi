        Physical Computing
        
Arduino Modulino® + NVIDIA Orin Nano

        
Software intelligence bridged with physical hardware. 7 sensors over a single Qwiic I2C
          cable.

      

      

        

          

            
💡

            

              
Pixels 0x6C

              
8× RGB LEDs — LC8822

              Status: idle 🔵 signal 🟣 trade 🟡 win 🟢 loss 🔴
            

          

          

            
🔊

            

              
Buzzer 0x3C

              
PKLCS1212E — Tone Generator

              Audio alerts for signals, trades, wins, errors
            

          

          

            
🔘

            

              
Buttons 0x7C

              
3× Push + LED

              A = trigger cycle · B = toggle mode · C = e-stop
            

          

          

            
🎛️

            

              
Knob 0x76

              
PEC11J — Rotary Encoder

              Real-time RSI threshold tuning
            

          

          

            
🧭

            

              
Movement 0x6A

              
LSM6DSOX — 6-axis IMU

              Tilt detection → auto-pause trading
            

          

          

            
🌡️

            

              
Thermo 0x44

              
HS3003 — Temp + Humidity

              Environment logging to ClawVault
            

          

          

            
📡

            

              
Distance 0x29

              
VL53L4CD — Time-of-Flight

              Proximity wake-up (<5cm triggers cycle)

        Get Started in 60 Seconds
        
Quick Start

      


      

        
⚡ One-Shot Install (curl / npm)

        
Install everything in a single command, OpenClaw/Hermes style.


        

          

            

              curl install
              
            

            

              

                
curl -fsSL https://raw.githubusercontent.com/x402agent/mawdbot-go/main/install.sh | bash

              

            

          


          

            

              with web console
              
            

            

              

                
curl -fsSL https://raw.githubusercontent.com/x402agent/mawdbot-go/main/install.sh | bash -s -- --with-web

              

            

          


          

            

              npm / npx flow
              
            

            

              

                
npx -y ./npm/mawdbot-installer --with-web

              

            

          

        


        
Installs: mawdbot, mawdbot-tui, optional
          mawdbot-web, and nano symlink.

      


      

        

          
1

          
Clone & Build

          

            

              
$ git clone https://github.com/x402agent/NanoSolana-tamaGOchi.git
$ cd NanoSolana-tamaGOchi
$ cp .env.example .env
$ make build

            

          

        

        

          
2

          
Configure

          

            

              
# Edit .env with your API keys
HELIUS_API_KEY=your_key_here
TELEGRAM_BOT_TOKEN=your_token
BIRDEYE_API_KEY=optional

            

          

        

        

          
3

          
Run

          

            

              
$ ./build/mawdbot daemon
🐹 TamaGOchi — A GoBot on Solana

🤖 Telegram: @YourBot connected
