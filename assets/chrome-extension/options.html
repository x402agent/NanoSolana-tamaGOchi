NanoSolana Browser Agent Relay

Relay browser tabs, manage wallet state, chat via gateway/Telegram, and trigger manual trades.

Getting started

If you see a red ! badge on the extension icon, the relay server is not reachable. Start NanoSolana browser relay on this machine, then click the toolbar icon again.

Docs: nanosolana.com

Connection + auth

Relay endpoint: http://127.0.0.1:<port>/.
Used by /health and /api/extension/* endpoints.

Security notes

The shared gateway secret can execute wallet/chat/trade actions. Keep this extension profile private and use trusted local or secured gateway endpoints.

For remote gateways, prefer HTTPS and private network routing (VPN/Tailscale).
