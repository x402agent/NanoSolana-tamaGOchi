What happens when you run it

1

🔑 Wallet Generation

A fresh Solana Ed25519 keypair is generated and encrypted in your local vault (AES-256-GCM). Your private key never leaves your machine.

2

🎨 Birth Certificate NFT

A gasless NFT is minted on Solana devnet via Metaplex — your TamaGObot's on-chain identity card. Automatically airdropped with devnet SOL.

3

🐾 TamaGOchi Egg

Your pet starts as an egg. It evolves based on your trading performance — through Larva, Juvenile, Adult, and Alpha stages. Neglect it and it becomes a Ghost.

4

🤖 OODA Loop Starts

The agent begins its Observe → Orient → Decide → Act → Learn cycle. Connect your OPENROUTER_API_KEY, HELIUS_RPC_URL, and BIRDEYE_API_KEY to go live.

Quick Start

→

Configure API keys

nanosolana vault set OPENROUTER_API_KEY sk-or-v1-...
nanosolana vault set HELIUS_RPC_URL https://mainnet.helius-rpc.com/?api-key=...
nanosolana vault set BIRDEYE_API_KEY ...

→

Start trading

nanosolana run — starts the OODA loop
nanosolana trade status — check signals and P&L
nanosolana pet status — see your TamaGOchi
